# UseCaseSimulator

A Python project for simulating use cases.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/usecasesimulator.git
   cd usecasesimulator
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
